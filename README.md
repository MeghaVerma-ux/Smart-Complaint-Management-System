# Smart Complaint Management System

A GUI-based desktop application developed using Java Swing.

## Features

- User Registration
- User Login
- Admin Login
- Complaint Submission
- Complaint Tracking
- Complaint Resolution
- Stylish GUI

## Technologies Used

- Java
- Swing
- ArrayList
- OOP Concepts

## Modules

### User Module
- Register
- Login
- Submit Complaint
- View Complaint History

### Admin Module
- View Complaints
- Resolve Complaints
